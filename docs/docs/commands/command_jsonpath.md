# jsonpath

Evaluate a JSONPath expression
## Aliases

* `jsonpath`
* `jspath`

## Synopsis

```shell
| jsonpath <[expression=]expression> <[field=]source field>[[dest=]dest field]
```

## Schema

```json
{
  "properties": {
    "{dest}": {
      "description": "Extracted field"
    }
  }
}
```
